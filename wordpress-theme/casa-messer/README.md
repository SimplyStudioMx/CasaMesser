# Tema Casa Messer

1. Sube la carpeta `casa-messer` a `wp-content/themes/` y actívala desde **Apariencia → Temas**.
2. En **Apariencia → Personalizar** edita foto, textos, botón de portada, tipografías y tamaño general de texto.
3. En **Secciones → Agregar nueva**, crea Catas, Experiencias, Menú, Eventos, Bodas, Eventos privados y La comida. Usa el editor para la información, asigna una **imagen destacada** y completa el bloque **Tarjeta y botón**.
4. Ordena las tarjetas con el campo **Orden** de cada sección.
5. En **Ajustes → Lectura**, elige una página como portada estática para utilizar `front-page.php`.

El tema incluye fotografías de respaldo en `assets/`; en producción sustitúyelas desde la Biblioteca de medios usando la imagen destacada de cada sección y la imagen de portada del Personalizador.
